## Description:
You go on a date with a very normal human person girl and normal human date things happen.
It's sort of a parody of those cheesy anime dating sims, where instead of a cute anime girl, you get this freak. Juno is actually an original character of mine from another story!

This project was more or less just an exploration of a specific library haha. I used this library called Ren'Py for this project, which is a staple tool in creating visual novels in Python. this is the same engine popular visual novels like 'Doki Doki Literature Club' and 'Slay the Princess' were made in.

When you launch the engine, the program gives you a templete for various files you need for it to work. The first one I dealt with is the 'Script.rpy' file which is the main file for controlling what scenes and events happen in the game. In that, I implemented the dialogue, character art, multiple choices and the ability to write your character name. it also dowloads multiple folders that the program will call to input into the game. This includes an "images" folder where I put all the sprites and backgrounds, and an "audio" folder, where I put the music and sound effects. There is also a folder and file for the GUI, but I didn't mess with it too too much as a lot of the default settings looked fine to me haha.

My initial plan was to have placehodlers for Juno's character art until I had time to put in the Actual art, which would have ben colored and more detailed. But as I tries to further deatil the art, I found that it just didn't have the same feel. I got so endeared to the placholder art, that I js decided to clean it up a bit and keep it! 

I've shown this game toa. few of my friends at this point and many of them were interested in tryong out this library and making one of their own! which makes me really happy, beacause this was genuinley a super fun program to work with. Programming is NOT my favorite thing, one could argue it's my least favorite thing! But this program kinda brought the fun back into it!
