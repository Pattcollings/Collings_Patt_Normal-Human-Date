Normal Human Date

## Repository
<https://github.com/Pattcollings/Collings_Patt_Normal-Human-Date.git>

## Description
A very short visual novel 'dating sim' where you're on a date with an alien who is (very badly) pretending to be a human.

## Features
- Feature 1
	- dialouge options the player can click on to influence the date positively or negatively
- Feature 2
	- The alien character reacts differently based on the player's choices with various sprites
- Feature 3 
	- Background audio to give the game some atmosphere


## Challenges
- how to add audio to the program for atmosphere
- adding multiple images that change based on the dialouge options
- how to have the player interact with things on the screen an not just in the terminal

## Outcomes
Ideal Outcome:
- The visuals are high quality
- the storyline changes based on the user's input
- the game can either have a good neutral or bad ending

Minimal Viable Outcome:
- the visuals exist 
- user input is acknowleged
- the game is able to end
